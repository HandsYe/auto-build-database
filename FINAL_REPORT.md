## 🚀 使用指南

### 快速开始
```bash
# 1. 克隆项目
git clone https://github.com/YOUR_USERNAME/biodeploy.git
cd biodeploy

# 2. 安装
pip install -e .

# 3. 验证
python scripts/verify_project.py

# 4. 测试
python scripts/test_complete.py

# 5. 使用
biodeploy --help
biodeploy list
biodeploy install ncbi_refseq_protein
```

### Python API
```python
from biodeploy import InstallationManager

manager = InstallationManager()
success = manager.install("ncbi_refseq_protein")
```
